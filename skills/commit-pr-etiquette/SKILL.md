# Commit & PR Etiquette

## Rules

- Do not add a Claude Code footer (e.g. `Co-Authored-By: ...` or "Generated with Claude Code") to commit messages or PR descriptions.
- When reviewing code or PRs, call out any Claude Code footer present in commit messages or PR descriptions and ask the author to remove it.
