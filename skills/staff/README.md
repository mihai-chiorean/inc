# /staff skill

Per-project agent staffing for Claude Code. See `SKILL.md`.

## Layout

```
skills/staff/
  SKILL.md                  # skill entry point (frontmatter + usage)
  README.md                 # this file
  docs/
    schemas.md              # manifest + lockfile + overlay + merged-file specs
  examples/
    lock.example.yaml       # sample .claude/staff/lock.yaml
    overlay.example.md      # sample .claude/staff/overlays/<id>.md
    config.example.yaml     # sample .claude/staff/config.yaml
```

The HR-repo root has:

```
agent.manifest.yaml         # canonical roster, generated by scripts/generate-manifest.py
scripts/generate-manifest.py
```
